# キンミヤレモン

## 食材

* 国産レモン 4個
* キンミヤ焼酎 900ml
* 氷砂糖 75g

## 作り方

1. `レモン`をよく洗い、皮をピーラーで薄く剥く **※[リモンチェッロ](https://hackmd.io/@GyTP6pUqQkKQUZiFBNIT2Q/HJq2iJGJv)で残った果肉を使うと良い**
2. `レモン`の果肉を1/8ぐらいにカットして瓶に入れる
3. `キンミヤ焼酎`を瓶に入れる
4. `氷砂糖`を瓶に入れる
5. 1ヶ月寝かせる
