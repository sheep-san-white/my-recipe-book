# ハンバーグ

## 食材

* A
    * 玉ねぎ（小〜中） 1玉
    * 生パン粉 130g
    * 牛乳 130ml
* B
    * 合いびき肉（豚・牛） 粗挽き 600g
    * マジックソルト 小さじ1
    * 茅乃舎 野菜だし 2パック
    * ナツメグ 小さじ1
    * チューブにんにく 3cm

## 作り方

1. 玉ねぎをあらみじん切りにする
2. `A`の具材をボウルで混ぜ、パン粉に牛乳を吸わせる
3. `A`のボウルに`B`の具材を入れてよく混ぜる
4. `120g`ずつ小判型に丸め、手の中で表裏空気を抜く
5. 冷蔵庫で30分冷やす
    * お昼にタネを作っておいて夕方焼くのでもOK
6. フライパンで両面焼き目がつくまで焼き、蓋をして極弱火で10分焼く
