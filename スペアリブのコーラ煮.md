# スペアリブのコーラ煮

## 食材

* スペアリブ 900g
* 塩コショウ 少々
* コーラ 700ml
* にんにく 2かけ
* 濃口醤油 大さじ3
* 酒 大さじ2
* みりん 大さじ1

## 作り方

1. `スペアリブ`に`塩コショウ`を振って暫くなじませる
2. フライパンで`スペアリブ`に焼き目をつける
3. 圧力鍋に`スペアリブ`と`コーラ`、`にんにく`、`酒`を入れ、アクを取りながら煮立たせる
4. 沸騰したら`濃口醤油`を追加し、圧力をかけて15分煮る
